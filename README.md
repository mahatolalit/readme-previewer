# 📖 Readme Previewer – Markdown Preview for VS Code

Preview your `README.md` (and other markdown files) inside **Visual Studio Code** with a clean layout that automatically adapts to your editor theme.

<img src="https://raw.githubusercontent.com/mahatolalit/assets/refs/heads/main/preview.gif" alt="Preview Screenshot" width="100%" />

---

## ✨ Features

- 🔹 **Adapts to VS Code Theme**: Light, Dark, and High Contrast are supported automatically.
- 🔹 **Syntax Highlighting**: Code blocks styled with Highlight.js, matching your active theme.
- 🔹 **Auto Refresh**: Updates when you save the file.
- 🔹 **Works Offline**: No external CDN requests; all styles are bundled within the extension.
- 🔹 **Secure Rendering**: Prevents execution of scripts inside Markdown, fully isolating the preview.

---

## 📦 Requirements

- **Visual Studio Code:** version `1.90.0` or higher.

---

## 🚀 Installation & Usage

1. **Install** from the [VS Code Marketplace](https://marketplace.visualstudio.com/items?itemName=mahatolalit.readme-previewer) 
   *(Alternatively, search for "Readme Previewer" in your VS Code Extensions view).*
2. Open any `README.md` (or any `.md`) file in VS Code.
3. Click on the **Preview** option from the footer status bar or Run the command **`Preview README`** from the Command Palette (`Ctrl+Shift+P`).
4. A side-by-side preview opens with full markdown rendering, following the active editor theme.

> **Note:** The preview’s appearance follows your current VS Code color theme automatically and works entirely offline.
